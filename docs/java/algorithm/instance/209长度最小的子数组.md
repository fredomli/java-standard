## 209.长度最小的子数组
### 题目链接 ：https://leetcode-cn.com/problems/minimum-size-subarray-sum/

*给定一个含有n个正整数的数组和一个正整数s,找出该数组中满足其和≥s的长度最小的连续子数组，并返回其长度。如果不存在符合条件的子数组，返回0。*

示例1：

> 输入： s = 7, nums = [2,3,1,2,4,3]  
> 输出： 2  
> 解释： 子数组 [4,3] 是该条件下的长度最小的子数组
