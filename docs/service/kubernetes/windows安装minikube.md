# windows安装 minikube

## 安装minikube
[minikube官方安装说明](https://minikube.sigs.k8s.io/docs/start/)

官方地址有详细的安装说明，按照官方安装说明一步步安装即可。

## 开始您的集群
从具有管理员权限的终端(但没有以root身份登录)运行:

```shell
minikube start
```
> 如果minikube启动失败，请参阅 [驱动程序](https://minikube.sigs.k8s.io/docs/drivers/) 页面，以获取设置兼容容器或虚拟机管理器的帮助。
