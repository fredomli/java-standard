# Learn Kubernetes Basics

> Kubernetes，也称为K8s，是一个用于自动化部署、扩展和管理容器化应用程序的开源系统。

它将组成应用程序的容器分组为逻辑单元，以便于管理和发现。Kubernetes基于15年在谷歌运行生产负载的经验，结合社区的最佳想法和实践。
