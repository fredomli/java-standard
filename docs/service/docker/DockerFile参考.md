# DockerFile参考
过读取Dockerfile中的指令，Docker可以自动构建映像。Dockerfile是一种文本文档，其中包含用户在命令行上可以调用的所有命令来组装图像。使用docker构建用户可以创建一个自动构建，它可以连续执行多个命令行指令。

本页面描述了在Dockerfile中可以使用的命令。阅读完本页后，请参考Dockerfile最佳实践以获得面向技巧的指南。
