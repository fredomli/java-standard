# Spring WebFlux（响应式）

## 概述

Spring框架中包含的最初web框架Spring web MVC是专门为Servlet API和Servlet容器而构建的。响应式堆栈web框架Spring WebFlux在5.0版本中被添加。它是完全无阻塞的，支持响应流回压，并在Netty、Undertow和Servlet 3.1+容器等服务器上运行。
