## VM安装

打开下载好的`.exe`文件，根据安装提示完成安装。步骤大致如下：

1. 进入安装的欢迎界面，下一步。
2. 进入协议界面，阅读协议并同意协议内容，下一步。
3. 选择安装方式，默认安装位置直接下一步，也可以自定义安装，指定安装位置。
4. 用户体验设置，包括启动时检查产品更新和加入客户体验计划两项，建议不选，直接跳过。
5. 最后直接默认安装，直到安装完成。

> VMware是收费产品，根据自己需要安装。
